#### 53.最大子数组和

##### 题目：

​	给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组**是数组中的一个连续部分。

**示例 1：**

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
```

**示例 2：**

```
输入：nums = [1]
输出：1
```

**示例 3：**

```
输入：nums = [5,4,-1,7,8]
输出：23
```

##### 标签：

​	数组，分治，动态规划

##### 思路：

​	这道题可以有三个思路，分别是最小前缀，动态规划和分治，我一个一个写我的思路

​	**最小前缀：**一个区间的值就是从头到区间最右端的和减去这个区间开始的前缀，那么这个数组中的最大子数组，就是最大的区间值减去这个区间中包含的最小前缀和，那么就从最开始遍历这个数组，设置三个值，其中一个值就是到当前遍历的位置的和，第二个值就是目前得到的最大子数组和，也就是返回的值，第三个值就是当前的最小前缀，然后在遍历的时候让第一个值一直加，然后每遍历到一个值的时候就用第一个值去减去最小前缀，用来更新第二个值，取最大的，在这之后更新最小前缀，是第一个值还是第三个值

​	**动态规划：**动态规划就是走一步看一步，且后一步不会影响前一步走的路，对于这道题，如果你按包含当前遍历值来进行动态规划，你会发现在遍历下一个值的时候就可能会更新前一个值，这是不对的，所以我们改一个思路，那就是动态规划以当前遍历值结尾的最大子数组和，所以每遍历到一个值时，你就判断是目前的前面最大值加上这个值更大，还是这个值更大，如果目前的前面最大值小于零，很显然就是这个大，以此来不断进行更新，就可以得出返回值。

​	**分治：**将他一直分，因为最大值不是在左区间就是在右区间，或者包含中间的两个值，那么你把他无限分到最小，这样的话就只需要返回当前区间值，左最大前缀和右最大后缀和，以及当前最大区间和，很显然合并时候区间值直接加就好，左最大前缀和就是之前的左大前缀和和之前的左区间和加上之前的右边的左最大前缀和，右最大后缀和就是之前的右边的最大后缀和和之前的右边的区间和加之前的左边的右后缀和，而最大值要么是左最大值，右最大值，要么是左后缀加右前缀。

##### 代码：

最小前缀和：

```Java
class Solution {
    public int maxSubArray(int[] nums) {
        int pre=0;
        int res=nums[0];
        int presum=0;
        for(int i=0;i<nums.length;i++){
            presum+=nums[i];
            res=Math.max(res,presum-pre);
            pre=Math.min(pre,presum);
            
        }

        return res;
    }
}
```

动态规划：

```Java
class Solution {
    public int maxSubArray(int[] nums) {
       int pre=nums[0];
       int max=nums[0];
       for(int i=1;i<nums.length;i++){
         pre=Math.max(nums[i],pre+nums[i]);
         max=Math.max(pre,max);
       }
       return max;
    }
}
```

分治：

```java
class Solution {
    // 返回值: [sum, lm, rm, max] = [区间和, 前缀最大值, 后缀最大值, 最大子数组和]
    int[] dfs(int[] nums, int l, int r) {
        if (l == r) {
            int t = Math.max(nums[l], 0);
            return new int[]{nums[l], t, t, t};
        }
        // 划分成两个子区间，分别求解
        int mid = l + r >> 1;
        int[] left = dfs(nums, l, mid), right = dfs(nums, mid + 1, r);
        // 组合左右子区间的信息，得到当前区间的信息
        int[] ans = new int[4];
        ans[0] = left[0] + right[0]; // 当前区间和
        ans[1] = Math.max(left[1], left[0] + right[1]); // 当前区间前缀最大值
        ans[2] = Math.max(right[2], right[0] + left[2]); // 当前区间后缀最大值
        ans[3] = Math.max(Math.max(left[3], right[3]), left[2] + right[1]); // 最大子数组和
        return ans;
    }
    public int maxSubArray(int[] nums) {
        int m = nums[0];
        for (int x : nums) m = Math.max(m, x);
        if (m <= 0) return m;
        return dfs(nums, 0, nums.length - 1)[3];
    }
}

```
