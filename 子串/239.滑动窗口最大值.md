#### 239.滑动窗口最大值

##### 标签：

​	队列，数组，滑动窗口，单调队列

##### 题目：

​	给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。

返回 *滑动窗口中的最大值* 。

 

**示例 1：**

```
输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
输出：[3,3,5,5,6,7]
解释：
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

**示例 2：**

```
输入：nums = [1], k = 1
输出：[1]
```

##### 思路：

​	这道题的思路对于知道单调队列的人不算难，不过我第一次做这道题可真是难住了，我尝试用前缀，也尝试定义一个数组，长度是k+1或者k+2，其中分别存K个滑动窗口的值用那个%k，最后是最大值和最大值索引，不过失败了

​	其实思路就是定义一个队列，让队列的最前端是目前的最大值，遍历数组时候，先判断删除的那个i-1是不是最大值（队列头），是的话就让队列头移除，因为是单调队列，之后判断加入的那个值和队列目前的尾部的大小判断，如果这个值大于队列尾部值，就把队列尾部值移除（while循环），理由就是因为新加的那个值在数组中的位置在他们的后边，如果这个值已经比他们大了，那后续的最大值肯定与他们无关，在这之后把那个值加到队尾，在滑动窗口形成后，每次遍历都需要把当前滑动窗口的最大值也就是队列头加入到返回的数组中

##### 代码：

​	

```Java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
       Deque<Integer> deque=new LinkedList<>();
       int[] res=new int[nums.length-k+1];

       for(int j=0,i=1-k;j<nums.length;j++,i++){
            if(i>0&&deque.peekFirst()==nums[i-1]){
                deque.removeFirst();
            }

            while(!deque.isEmpty()&&deque.peekLast()<nums[j]){
                deque.removeLast();
            }

            deque.addLast(nums[j]);
            if(i>=0){
                res[i]=deque.peekFirst();
            }
       }
       return res;
    }
}
```
