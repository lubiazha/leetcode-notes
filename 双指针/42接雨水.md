#### 42.接雨水

##### 题目：

​	给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

 

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/10/22/rainwatertrap.png)

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
```

**示例 2：**

```
输入：height = [4,2,0,3,2,5]
输出：9
```

#####  思路：

​	最开始就是想按列求，这道题的本质其实就是每一列能接的雨水取决于他的左右两边更矮的那个最高点是否比当前列高，如果比当前列高的话，这一列能存的雨水不正是那个更矮的最高点和当前列的高度差吗，因此，只需要先定义两个数组，让这两个数组分别代表某行两端的最高点，之后再从左到右或者从右往左遍历数组，即可求出

​	不过后续我又想了一下，那就是能不能出去两个数组呢，首先，因为是从左到右遍历的，所以我就想在遍历的过程中，用一个变量来代替左边的数组，这样是简化的第一步，那我又想能否去除右面的数组呢，答案是显而易见的，因为左边最大的变量的更新是判断当前列的左边列和最大值的比较，我们能够简化只是因为我们从左遍历，使用的是i，那如果我们从右遍历对于右面的也是可以同样处理的，所以如果我们两端同时遍历呢，如何来判断是从左面遍历还是右面遍历就是最大的问题，事实上，从哪端遍历就是取决于哪端最大的值更小，这样才可以求出存水量，所以，我们可以得出只需要判断当前的左边列(可能更新左端最大)与右边列（可能更新右段最大）的值就好了，哪个更小就走那边。最大的遍历只需要遍历出足够的列数就好

​	其实这道题用栈也可以求解，不过效率上不如双指针，我就不赘述了



##### java代码：	

```Java
class Solution {
    public int trap(int[] height) {
    
        int sum=0;
        int left_max=0;
        int right_max=0;
        int left=1;
        int right=height.length-2;

        for(int i=1;i<height.length-1;i++){
           if(height[left-1]<height[right+1]){
                left_max=Math.max(left_max,height[left-1]);
                if(left_max>height[left]){
                    sum+=(left_max-height[left]);
                }
                left++;
           }else{
                right_max=Math.max(right_max,height[right+1]);
                if(right_max>height[right]){
                    sum+=(right_max-height[right]);
                }
                right--;
           }
        }
        return sum;
    }
}
```
