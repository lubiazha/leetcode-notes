#### 438.找到字符串中所有字母异位词

##### 题目：

​	给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

 

#####  示例 1:

```
输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。
```

 **示例 2:**

```
输入: s = "abab", p = "ab"
输出: [0,1,2]
解释:
起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。
```

##### 思路：

​	看过我的上一题的都知道，滑动窗口有一个固定的思路，并且基本每一个题都有更好一点的进阶解法，我就直接说进阶解法，模板思路可以看我的上一题。

​	这个进阶解法就是用两个26大小的数组，是比哈希表要更简单一些的，然后把需要找的遍历，存到这个数组里，int型的，因为都是小写，值就是出现的次数，然后只需要定义左右指针，右指针在for循环就行，之后就是遍历一个右指针，然后就把值加进去，然后循环判断对于这个值在窗口中的数量是否比需要的多，需要的多的话就把左边往后坐，有些人在这里就会有疑问，那就是只判断这一个值吗，实则不然，如果这个循环退出了，那就说明以这个值为最右端的一个字串里，目前这个值是符合需要的值的，之后判断这个串的长度和p的长度，如果等长，那很显然就是其中要的一个解，因为以他为最右端的子串中的每一个值在这个串中都只有符合需要的值，因为每一次遍历都要判断一下，因此可以判断

##### 代码：

​	通用模板：

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        Map<Character,Integer> window=new HashMap<>();
        Map<Character,Integer> need=new HashMap<>();
        for(int i=0;i<p.length();i++){
            need.put(p.charAt(i),need.getOrDefault(p.charAt(i),0)+1);
        }
        int left=0;
        int right=0;
        int valid=0;
        List<Integer> res=new ArrayList<>();
        while(right<s.length()){
            char c=s.charAt(right);
            right++;
            if(need.containsKey(c)){
                window.put(c,window.getOrDefault(c,0)+1);
                if(window.get(c).equals(need.get(c))){
                    valid++;
                }
            }
            while(right-left>=p.length()){
                if(valid==need.size()){
                    res.add(left);
                }
                char c1= s.charAt(left);
                left++;
                if(need.containsKey(c1)){
                     if(window.get(c1).equals(need.get(c1))){
                        valid--;
                    }
                    window.put(c1,window.get(c1)-1);
                }
            }
        }
        return res;
    }
}
```

​	优化版本：

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        int n = s.length(), m = p.length();
        List<Integer> res = new ArrayList<>();
        if(n < m) return res;

        int[] pCnt = new int[26];
        int[] sCnt = new int[26];

        for(int i = 0; i < m; i++){
            pCnt[p.charAt(i) - 'a'] ++;
        }

        int left = 0;
        for(int right = 0; right < n; right++){
            int curRight = s.charAt(right) - 'a';
            sCnt[curRight]++;
            while(sCnt[curRight] > pCnt[curRight]){
                int curLeft = s.charAt(left) - 'a';
                sCnt[curLeft]--;
                left++;
            }
            if(right - left + 1 == m){
                res.add(left);
            }
        }
        return res;
    }
}
```
