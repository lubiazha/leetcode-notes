#### 3.无重复字符的最长字串

##### 题目：	

​	给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**示例 1:**

```
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
```

**示例 2:**

```
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

##### 思路：

​	对于这道题来讲的话，其实是有两种个思路，一种是滑动窗口的通用模板，一种是适用于这道题的优化。

​	先说通用模板，对于滑动窗口来讲，通用模板是这个

```
class{
	main(String s,String p){
		先把p遍历，挨个字符加入一个need的哈希表里，键值对是Character,Integer;
		遍历s
		while(right<s.length){
		把s的当前字符加入到window哈希表
		right++;
		随后判断window里面的当前值的值是否等于need里面的值,等于的话就让vaild变量加一
		
		while(判断vaild的值是否等于p的长度){
				在这里面移动左边的指针，然后进行一系列值的更新
			}
		}
	}		
}
```

对于这道题也是如此，只不过其中不会有need这个哈希表和valid这个值，且第二个while循环里面直接判断右指针的值对应的值是否大于一，然后移动左指针

​	第二种办法对于这道题就很简单了，第一个循环不变，并且不需要第二个循环，在把右指针的值加入到哈希表之前进行一个if判断，如果当前哈希表里已经有这个值了，就把左指针跳到这个值的值的下一位start=Math.max(map.get(c),start);，这里面的哈希表的值是这个字符在字符串里面的位置的下一位，随后算出子串长度最大值，然后将当前字符更新到哈希表里

##### 代码：

通用模板：

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
      char[] sArray=s.toCharArray();
      Map<Character,Integer> map=new HashMap<>();
      int left=0;
      int max=0;
      for(int right=0;right<sArray.length;right++){
        map.put(sArray[right],map.getOrDefault(sArray[right],0)+1);
        while(map.getOrDefault(sArray[right],0)>1){
            char c=sArray[left];
            map.put(sArray[left],map.getOrDefault(sArray[left],0)-1);
            left++;
        }
        max=Math.max(right-left+1,max);
      }

        return max;
    }
    }
```

优化解析：

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
     int n=s.length(),ans=0;
     Map<Character,Integer> map=new HashMap<>();
     for(int end=0,start=0;end<n;end++){
        char c=s.charAt(end);
        if(map.containsKey(c)){
            start=Math.max(map.get(c),start);
        }
        ans=Math.max(ans,end-start+1);
        map.put(s.charAt(end),end+1);
     }
     return ans;     
    }
}

```
